📌 Loan Default Prediction using Machine Learning
This project focuses on building an end-to-end machine learning classification system to predict loan default using customer banking data. The objective is to identify whether a borrower is likely to default on a loan based on various financial and demographic features.
The project begins with Exploratory Data Analysis (EDA), where histograms and box plots are used to understand data distribution and detect outliers. This is followed by comprehensive data preprocessing, including handling missing values, removing duplicates, encoding categorical variables, and applying feature scaling to prepare the data for modeling.
Two classification algorithms are implemented: Support Vector Machine (SVM) with different kernel functions (linear, polynomial, and radial basis function) and K-Nearest Neighbors (KNN). These models are trained and evaluated to determine their effectiveness in predicting loan default.
Model performance is assessed using standard evaluation metrics such as precision, recall, F1-score, and ROC-AUC. The project demonstrates a complete machine learning pipeline from raw data preprocessing to model evaluation, following best practices in data handling and model development.
